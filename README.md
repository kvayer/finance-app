Finance Tracker

Веб-приложение для учета личных финансов.
Разработано на чистом Go (Standard Library) с использованием архитектуры MVC.

Запуск

Требования

Docker

Docker Compose

Команды

# Запуск приложения и базы данных
make run

# Остановка
make stop


Приложение будет доступно по адресу: http://localhost:8080
